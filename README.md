================================================================================
                            وصف البرنامج
================================================================================

برنامج متخصص لإدارة الحسابات المالية لشركات التنقيب، مع التركيز على:

- إدارة الكولات (الفرق العاملة)
- إدارة حسابات العمال
- إدارة خزينة الشركة
- متابعة العمليات المالية وإنشاء التقارير

================================================================================
                        المتطلبات الأساسية
================================================================================

نظام التشغيل:
- Windows 10 أو أحدث

المتطلبات البرمجية:
- Python 3.10 أو أحدث
- PySide6
- SQLite3 (مدمج مع Python)

================================================================================
                        طريقة التثبيت
================================================================================

1. تثبيت Python 3.10+ من الموقع الرسمي:
   https://www.python.org/downloads/

2. تثبيت المكتبات المطلوبة:
   pip install PySide6

3. تشغيل البرنامج:
   py main.py

================================================================================
                        الميزات الرئيسية
================================================================================

📋 إدارة الكولات:
   - إضافة كولة جديدة (اسم، مسؤول، عمال)
   - تعديل بيانات الكولة
   - حذف كولة نهائياً
   - عرض جميع الكولات في جدول

👷 إدارة العمال:
   - إضافة عامل جديد (مع اختيار الكولة)
   - إضافة عمال دفعة واحدة عند إنشاء الكولة
   - عرض جميع العمال في جدول

💰 العمليات المالية:
   - تسجيل عمليات مالية جديدة
   - اختيار العمال المشاركين في العملية
   - حساب تلقائي (50% شركة، 50% كولة)
   - خصم الزوادة تلقائياً
   - عرض ملخص العملية قبل الحفظ

🏦 الخزينة:
   - عرض رصيد الخزينة الحالي
   - عرض حركات الخزينة
   - إضافة سلفة لعامل
   - استرداد سلفة (قيد التطوير)
   - صرف مستحقات عامل (قيد التطوير)

📊 لوحة المعلومات:
   - عرض إحصائيات فورية (عدد الكولات، العمال، العمليات، رصيد الخزينة)
   - عرض آخر العمليات المسجلة
   - عرض الكولات النشطة

⏰ الوقت والتاريخ:
   - عرض الوقت والتاريخ بشكل حي في شريط العنوان

================================================================================
                        قاعدة البيانات
================================================================================

الجداول:
1. Colas          - الكولات
2. Workers        - العمال

================================================================================
                        كيفية الاستخدام
================================================================================

1. إضافة كولة جديدة:
   - اذهب إلى تبويب "📋 الكولات"
   - أدخل اسم الكولة واسم المسؤول
   - اكتب أسماء العمال (كل اسم في سطر)
   - اضغط "➕ إضافة كولة مع العمال"

2. إضافة عامل بشكل مستقل:
   - اذهب إلى تبويب "👷 العمال"
   - أدخل اسم العامل واختر الكولة
   - اضغط "➕ إضافة عامل"

3. تسجيل عملية مالية:
   - اذهب إلى تبويب "💰 العمليات المالية"
   - اختر الكولة والتاريخ
   - أدخل المبلغ والزوادة (إن وجدت)
   - اختر العمال المشاركين
   - راجع الملخص ثم اضغط "💰 تسجيل العملية"

4. عرض الخزينة:
   - اذهب إلى تبويب "🏦 الخزينة"
   - عرض الرصيد الحالي والحركات
